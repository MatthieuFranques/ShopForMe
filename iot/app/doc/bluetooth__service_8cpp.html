Detailed Description

Implementation of Bluetooth service for ESP32.

This file contains the initialization of the Bluetooth Low Energy (BLE) service for an ESP32, handling of device connections and disconnections, as well as sending notifications.

◆ initBluetooth()

void initBluetooth ( )

Initializes Bluetooth on the ESP32.

This function configures and starts the BLE server on the ESP32. It also creates a BLE characteristic (read and notify) within a service specified by its UUID.

Note
The BLE server is started and begins advertising its availability.
