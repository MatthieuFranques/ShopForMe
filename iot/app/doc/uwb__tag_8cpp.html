Detailed Description

Source file for handling UWB (Ultra Wideband) tag operations.

This file contains functions to initialize the UWB tag, manage the tag's range measurements, and handle devices in the range. It interfaces with the DW1000 device to handle distance measurements, devices, and the package construction for sending the data via Bluetooth.

void handleTagLoop ( )

Handles the tag loop.

This function is called periodically to handle UWB ranging logic. It checks the time interval and logs the distances to the anchors. The data is then sent to the data handler for packaging and sending via Bluetooth.

Note
This function should be called in the main loop to continuously handle the tag's behavior.
